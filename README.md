> Open Charge Point Interface implementation in Go.

## Installation

Go version 1.24+ is required.

```sh
go get github.com/si3nloong/ocpi-go
```
## Features and supported versions
### Supported versions

-   [x] OCPP 2.1.1
-   [x] OCPP 2.2.1
-   [x] OCPI 2.3.0
-   [ ] OCPI 3.0.0
